Integrate Fikiri chatbot, lead capture, and CRM features into your WordPress site.

== Description ==

Fikiri Integration plugin allows you to easily add Fikiri's AI-powered chatbot, lead capture forms, and CRM features to your WordPress website.

**Features:**

* AI-powered chatbot widget
* Lead capture forms
* Contact forms
* Automatic CRM integration
* Customizable themes and positions
* Shortcode support
* Gutenberg block support

**Easy Setup:**

1. Install and activate the plugin
2. Go to Settings → Fikiri
3. Enter your API key from the Fikiri dashboard
4. Configure chatbot settings
5. Add shortcodes or blocks to your pages

**Shortcodes:**

* `[fikiri_chatbot]` - Display chatbot widget
* `[fikiri_lead_capture]` - Display lead capture form
* `[fikiri_contact_form]` - Display contact form

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/fikiri-integration/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings → Fikiri and configure your API key
4. Add shortcodes or blocks to your pages

== Frequently Asked Questions ==

= Do I need a Fikiri account? =

Yes, you need a Fikiri account and API key. Get one at https://app.fikirisolutions.com

= How do I get my API key? =

Log into your Fikiri dashboard, go to Settings → API Keys, and create a new key.

= Can I customize the chatbot appearance? =

Yes, you can customize the theme, position, and title in the plugin settings or via shortcode parameters.
