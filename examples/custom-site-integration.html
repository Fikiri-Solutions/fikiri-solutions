REST API Example (Alternative Method)

You can also use the REST API directly without the SDK:


fetch('https://api.fikirisolutions.com/api/webhooks/leads/capture', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-API-Key': 'fik_your_api_key_here'
    },
    body: JSON.stringify({
        email: 'user@example.com',
        name: 'John Doe',
        source: 'custom_website'
    })
})
.then(response => response.json())
.then(data => console.log(data));

Setup Instructions

  1. Get your API key from the Fikiri dashboard
  2. Replace fik_your_api_key_here with your actual API key
  3. Customize the chatbot theme and position if needed
  4. Test the integration on your website

Note: Make sure to set debug: false in production.
